安装”GBK Encoding Support”插件

Sublime Text 2默认不支持gbk编码文件，打开后会出现乱码，安装”GBK Encoding Support”插件使其支持GBK编码。

按command+Shift+P快捷键，打开命令行模式，输入Install Package

回车后，将会弹出

输入 “GBK Encoding Support” ，然后回车，将会自动安装这个插件。
还可以再安装“ConvertToUTF8”插件，用来转换文件编码到utf8。
